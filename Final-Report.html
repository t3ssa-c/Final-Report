Data

The 51-item STARS was partitioned into its two theoretically established domains: the affective sub-domain (items 1-23) and cognitive sub-domain (items 24-51). Partial correlation networks were estimated for both sub-networks, as well as the global network, and subsequently converted into undirected, weighted igraph objects for analysis.

# load the data
data <- read.csv('stars-data_132.csv', header = T)
# split into sub-networks and full network
affective <- data |> select(item_1:item_23) # affective sub-network
cognitive <- data |> select(item_24:item_51) # cognitive sub-network
all <- data |> select(item_1:item_51)  # full network
# create partial correlation matrix for each sub-network and full network
net_affective <- estimateNetwork(affective, default = "pcor", 
                                threshold = 'sig') # affective partial correlation matrix
## Estimating Network. Using package::function:
##   - qgraph::qgraph(..., graph = 'pcor') for network computation
##   - psych::corr.p for significance thresholding
net_cognitive <- estimateNetwork(cognitive, default = "pcor", 
                                threshold = 'sig') # cognitive partial correlation matrix
## Estimating Network. Using package::function:
##   - qgraph::qgraph(..., graph = 'pcor') for network computation
##   - psych::corr.p for significance thresholding
net_g <- estimateNetwork(all, default = "pcor", 
                         threshold = 'sig') # full network
## Estimating Network. Using package::function:
##   - qgraph::qgraph(..., graph = 'pcor') for network computation
##   - psych::corr.p for significance thresholding
# convert all to igraph objects for analysis
g_affective <- graph_from_adjacency_matrix(abs(net_affective$graph), 
                                           mode="undirected", weighted = TRUE)
g_cognitive <- graph_from_adjacency_matrix(abs(net_cognitive$graph), 
                                           mode="undirected", weighted = TRUE)
g <- graph_from_adjacency_matrix(abs(net_g$graph), 
                                 mode="undirected", weighted = TRUE)

Hypothesis 1: Internal Cohesion VS Path Efficiency

Does the cognitive sub-network exhibit greater internal cohesion, whereas the affective sub-system facilitate greater path efficiency?

Internal cohesion was tested by comparing macro-level metrics including average node strength, edge density, transitivity and degree assortativity; which mathematically assesses the presence of triads in the sub-network. The cognitive components of statistics anxiety are thus expected to exhibit significantly greater values for internal cohesion, presenting structurally as a clustered core of closed triangles, than affective items.

Path efficiency was evaluated using the Average Shortest Path Length (ASPL), where shorter paths indicate greater transmission efficiency. Accordingly, the affective sub-network is expected to exhibit a shorter ASPL than the cognitive sub-network, facilitating faster contagion.

Hypothesis 2: Network Degradation Simulation

Is structural disintegration, measured by decrease in global edge density, more efficacious from targeted “keystone” symptom deactivation than random symptom alleviation?

Keystone symptoms were defined as the top five nodes possessing the highest eigenvector centrality within the global network. To ensure fair comparison, five random nodes were deactivated.

cat("\n--- DEGRADATION SIMULATION (REMOVING 5 NODES) ---\n")
## 
## --- DEGRADATION SIMULATION (REMOVING 5 NODES) ---
baseline_density <- edge_density(g)

# Plot Network Before Degradation
V(g)$label <- gsub("^(i_|it_)", "", V(g)$name) # remove 'i_' and 'it_' for presentation and readability
plot(g, 
    vertex.color = "lightpink", 
    edge.color = "mediumpurple", 
    vertex.label.color = "black",
    vertex.size = 16,
    vertex.shape = 'circle',
    main = "Pre-Removal")

# Simulation A: Targeted Removal (Highest Strength)
ec <- eigen_centrality(g)$vector
targeted <- names(sort(ec, 
                       decreasing = TRUE)[1:5]) # identify top 5 nodes with highest ec 
g_targeted <- delete_vertices(g, targeted) # deactivate these nodes
density_targeted <- edge_density(g_targeted) # new density
difference_targeted <- baseline_density - density_targeted
print(difference_targeted)
## [1] 0.02080136
# Plot Targeted Degradation Network
V(g_targeted)$label <- gsub("^(i_|it_)", "", V(g_targeted)$name) # remove 'i_' and 'it_' for presentation and readability
plot(g_targeted, 
    vertex.color = "lightpink", 
    edge.color = "mediumpurple", 
    vertex.label.color = "black",
    vertex.size = 16,
    vertex.shape = 'circle',
    main = "Targeted Removal")

# Simulation B: Random Removal
set.seed(1)

Exploratory Analysis: Micro-Level Symptom Profiling

To translate these macro-level findings into actionable clinical targets, a post-hoc micro-level analysis was performed. Node strength was computed to identify the primary drivers (hubs) of anxiety within each domain, and betweenness centrality was calculated to locate the critical bridges across the network.

# Node Strength (Central Hubs)
strength_affective <- strength(g_affective)
strength_cognitive <- strength(g_cognitive)
# Betweenness Centrality (Structural Bridges)
betweenness_affective <- betweenness(g_affective, directed = FALSE, 
                                     weights = 1/abs(E(g_affective)$weight))
betweenness_cognitive <- betweenness(g_cognitive, directed = FALSE, 
                                     weights = 1/abs(E(g_cognitive)$weight))

# Top Three of Affective Sub-network
top_affective <- head(sort(strength_affective, decreasing = TRUE), 3)
print(top_affective)
